# WuWa Character Header — เลือกโหลดแค่อันเดียว (install ONLY ONE)

ทั้ง 4 ไฟล์ใช้แท็กเดียวกัน `[WCHAR|filename|Name|#hex|element · region]` — เลือกดีไซน์ที่ชอบแล้ว import แค่ไฟล์เดียว:

| ไฟล์ | ดีไซน์ |
|---|---|
| `WS_Header_A_ResonanceBar_Regex.json` | แผงกระจก กรอบเฉียง + อีควอไลเซอร์เคลื่อนไหว + ขอบทอง |
| `WS_Header_B_TacetMark_Regex.json` | รูปกลม กรอบข้าวหลามตัดหมุน (Tacet) + ชื่อ Cinzel |
| `WS_Header_C_GoldPlate_Regex.json` | แผ่นตัดมุม + แท็กธาตุ + เส้นทองคู่ (ดีฟอลต์) |
| `WS_Header_D_MinimalWave_Regex.json` | เรียบง่าย ใกล้ Global + เส้นคลื่นเรโซแนนซ์ |

`filename` = ชื่อไฟล์รูปตัวละคร (เช่น `changli`) — regex จะสร้าง URL ให้เอง. ทุกตัวมีอยู่ในลอร์บุ๊ค (WuWa • Character Portrait Links).
