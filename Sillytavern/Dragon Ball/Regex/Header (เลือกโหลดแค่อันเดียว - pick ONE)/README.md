# Dragon Ball Header — install ONLY ONE

All 4 use the same tag `[DBCHAR|path|Name|#hex|region|power]`. Import one design:

| File | Design |
|---|---|
| `DB_Header_A_ScouterHUD_Regex.json` | Green Frieza-saga scouter readout · power level · animated signal bars |
| `DB_Header_B_KiNameplate_Regex.json` | Pulsing ki aura · italic gradient name · full-width ki bar |
| `DB_Header_C_DragonBallPlate_Regex.json` | Orange gi plate · spinning 4-star dragon ball · hex-tinted name underline |
| `DB_Header_D_BattleCard_Regex.json` | Dokkan/Legends game-card · type tag · rarity stars · power meter |

`path` = character image under `Images/` without `.jpg` (e.g. `son_goku_super/06_super_saiyan_blue`). `power` is optional.
