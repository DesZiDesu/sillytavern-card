# Dragon Ball Character Monologue — install ONLY ONE

A character's own inner thoughts. Tag `[DBTHINK|Name|#hex|text]`. Renders **above** the header. Import one:

| File | Design |
|---|---|
| `DB_Monologue_A_MangaThought_Regex.json` | Cream + black ink thought bubble with trailing dots (pairs with Dialogue A) |
| `DB_Monologue_B_BattleThought_Regex.json` | Dark band, dashed hex accent = thinking (pairs with Dialogue B) |

This is for a **character** thinking. For scene narration with nobody speaking, use the **Narrator** folder instead.
