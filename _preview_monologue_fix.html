Before — Name label as wide flex header

The old header put the name on its own row inside the box with a diamond marker and a long dashed line. On narrow screens the header eats space and pushes the italic monologue text down or out of view.

After — Compact corner tag

The name is now a tiny caption tab hanging off the top-left corner of the box. It never blocks or shortens the monologue text, truncates cleanly if the name is long, and stays out of the way on both mobile and desktop.
