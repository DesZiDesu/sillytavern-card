# คัมภีร์ศิลป์ · Codex Velmyra — Art Bible
### แนวทางการสร้างภาพตราเผ่าและตัวละคร ทั้งเจ็ดสายโลหิตแห่งเผ่าอสูร

เอกสารนี้ใช้เป็นแนวทางเจนภาพให้ทั้งเซ็ตดู **เป็นโลกเดียวกัน** — โทนเดียว สไตล์เดียว
แต่ละเผ่ามี "สีประจำเผ่า" ที่ตรงกับสีบนเว็บ เพื่อให้ภาพที่เจนมากลมกลืนกับหน้าเว็บทันที

---

## ⚑ สไตล์รวมของทั้งโลก (Shared Style)

**โทนโลก:** dark high-fantasy medieval, ขลัง หรูหรา สง่างาม (แนว Mushoku Tensei / Berserk-lite)
**พื้นหลังหลัก:** ดำ-น้ำตาลเข้ม (charcoal / obsidian) + แสงทองอ่อนๆ
**ทองประจำคัมภีร์:** `#c9a227` (aged gold) — ใช้เป็นกรอบ/ตัวอักษร/ไฮไลต์ทุกภาพ

**Base style prompt (ต่อท้ายได้ทุกภาพ):**
```
dark high fantasy, painterly semi-realistic anime, muted cinematic palette,
volumetric candlight, aged gold accents, ornate medieval, intricate detail,
matte finish, artstation, 8k --ar 2:3
```
**Negative prompt (กันภาพเพี้ยน):**
```
lowres, extra fingers, bad anatomy, watermark, text, signature, oversaturated,
modern clothing, plastic skin, blurry, jpeg artifacts
```

### สไตล์ตราเผ่า (Emblem) — ให้ทั้ง 7 อันเป็นชุดเดียวกัน
เฟรมเดียวกันทุกเผ่า เปลี่ยนแค่สัญลักษณ์กลางกับสี:
```
heraldic sigil emblem, circular gold medallion with thin aged-gold filigree ring,
single-color line-art glyph in the center, dark obsidian background, symmetrical,
engraved etched look, clean vector heraldry, no text, centered --ar 1:1
```

## I · อิลวาริส — Ilvaris
**ผู้เดินสองโลก · สีประจำเผ่า:** `#93c58b` (sage green + gold)

**ตราเผ่า:** ใบหู-ใบไม้เรียวแหลม (ลูกผสมระหว่างหูแหลมกับใบไม้) ผ่ากลางด้วยเส้นก้านใบ
มีลายเส้นเลือดใบไม้แตกออกด้านข้าง และดาวเล็กดวงเดียวที่ปลายยอด — สื่อถึง "ผู้เชื่อมสองโลก" และการเป็นนักปราชญ์/ผู้จดบันทึก
```
Emblem prompt:

split by a central vein, delicate leaf-vein lines, a small star at the tip,
sage-green line-art on gold medallion, thin filigree ring, dark background,
symmetrical engraved heraldry, no text --ar 1:1
```

**รูปตัวละคร:** ดูเหมือนมนุษย์ทุกอย่าง ผิวปกติ ใบหน้าคมสุขุม เผยเพียง **ปลายหูแหลมสั้น** (สั้นกว่าเอลฟ์) โผล่พ้นเรือนผม แต่งกายแบบทูต/นักปราชญ์ยุคกลาง เสื้อคลุมผ้าดีปักลายทอง มือถือหนังสือหรือปากกาขนนก แววตาสงบเหมือนกำลังอ่านใจคน
```
Portrait prompt:
elegant scholar-diplomat, human-like face, calm knowing eyes, short pointed ears
peeking through hair (shorter than an elf's), refined medieval envoy robes with
gold embroidery, holding a leather tome and quill, muted candlelit library,
sage-green and gold palette, dark high fantasy, painterly semi-realistic anime --ar 2:3
```

## 📐 เคล็ดลับให้ทั้งเซ็ตเข้าชุดกัน
1. **ตราเผ่าทั้ง 7** — ใช้ base emblem prompt เดียวกัน เปลี่ยนแค่ "สัญลักษณ์กลาง + สี" จะได้เหรียญตราชุดเดียวกันเป๊ะ เจนที่ `--ar 1:1`
2. **รูปตัวละครทั้ง 7** — ล็อกมุมกล้อง/แสง/พื้นหลังให้คล้ายกัน (candlelit, dark background, ครึ่งตัว) เจนที่ `--ar 2:3` จะเรียงบนเว็บสวย
3. **สีประจำเผ่า** — คุมให้เด่นในแต่ละภาพ แต่มีทองอ่อนๆ ร่วมทุกภาพ = เอกภาพของโลก
4. ส่งภาพกลับมา ผมจะทำ **ช่องใส่ภาพ (image slot)** บนเว็บให้ — เอาตราเผ่าไปแทนโลโก้ SVG และเพิ่มรูปตัวละครข้างข้อมูลแต่ละเผ่าได้เลย
