# Honkai: Star Rail — Character Icons

Face-focus character icons (portrait style) for every playable unit, from launch to the newest arrivals.

## Contents

| Item | Description |
|------|-------------|
| `icons/` | 92 character icons — transparent PNG, 256×256, named `NNN_Name.png` (numbered by release order) |
| `manifest.json` | Metadata for every icon: index, name, Combat Type, rarity, file path |
| `index.html` | Self-contained browsable gallery — searchable by name, filterable by Combat Type. Open it directly in any browser (all images are embedded, no external files needed). |

## Gallery

`index.html` is fully self-contained (icons embedded as WebP data URIs), so it renders on any static host or straight from disk. It requires no login and no build step.

## Coverage

All 92 currently available playable characters, including every Trailblazer variant (per element, M/F) and the Fate/stay night collab units (Saber, Archer).

Not yet included: **Rin Tohsaka** and **Gilgamesh** — Fate collab Part 2, released after these assets were gathered.

## Source & notes

- Icons come from the [StarRailRes](https://github.com/Mar-7th/StarRailRes) community asset index, which mirrors the in-game roster.
- Names for the newest datamined units may be provisional and can change on official release.
- Artwork is the property of HoYoverse. These icons are collected for personal/reference use.
