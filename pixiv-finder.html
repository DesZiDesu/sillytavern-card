Pixiv Finderpersonal search & download via your own proxy

One-time setup

  1. Create a free account at dash.cloudflare.com
  2. Workers & Pages → Create → Worker → paste the contents of _pixiv_worker.js (in this repo) → Deploy
  3. Copy the worker URL (https://<name>.<acct>.workers.dev) and paste it below
  4. Required: add a worker secret PIXIV_SESSION = your pixiv.net PHPSESSID cookie. Pixiv blocks search without a logged-in session, so this is needed even for safe mode. (In Cloudflare: your worker → Settings → Variables and Secrets → Add → type Secret.)
  5. Optional: ACCESS_KEY secret (any password) locks your worker — enter the same value in Access key below. Leave blank if you skip it.

Personal use only — this talks to Pixiv through your worker with your optional session. Keep the worker URL private, keep volume low, and don't re-share artists' work. Session cookies expire after a few weeks; paste a fresh one if R-18 stops working.
